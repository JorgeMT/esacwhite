Wireless:
Quais os SSID's da rede wireless da ESAC?

Existem dois SSID's na rede wireless da ESAC : eduroam-guest e eduroam.

O SSID eduroam-guest é uma rede de acesso livre onde se encontram todas as configurações de acesso ao SSID principal eduroam. Apenas este último permite o acesso à Internet e está dotado de encriptação e acesso restrito.



Qual o protocolo de autenticação suportado na ESAC?

Os utilizadores da ESAC apenas têm disponível o PEAP - MS-CHAP-V2. Os utilizadores visitantes devem consultar a sua Instituição para receber esta informação.



Qual a configuração DHCP do cliente?

A sua configuração de rede deverá estar em modo automático, uma vez que os parâmetros de rede são enviados por DHCP.



Quais os sistemas operativos suportados para 802.1x/PEAP?

O 802.1x é suportado nativamente em Windows XP, Vista, MAC OS X 10.3. e versões mais recentes. Os restantes sistemas podem usar um cliente comercial.



Para que serve o certificado que tenho de instalar?

O certificado é um método de resguardo para o cliente de rede sem fios. Quando este se liga a um Ponto de acesso sem fios tem início um processo de autenticação com base no certificado. Este processo faz o cliente de rede sem fios assegura-se que a rede a que este se liga pertence realmente ao eduroam. Deste modo o cliente de rede sem fios com um certificado instalado, tem sempre a certeza que fica ligado a um ponto de acesso válido e não um outro qualquer pirata com o intuito de furtar palavras-chave.



Porque é que a minha placa, que funciona bem em casa e nos Hotspots, não funciona no eduroam?

Os pontos de acesso da ESAC têm de obedecer a normas de segurança regulamentadas no projecto eduroam. Na maior parte das vezes essas normas de segurança não são as vulgares e inseguras utilizadas em Hotspots ou redes domésticas. O método utilizado para estabelecer uma ligação de rede sem fios da ESAC é o mais seguro possível comummente praticável. É normal muitas das placas de rede sem fios não utilizarem os procedimentos mais recentes. É por este facto que é necessário fazer a actualização dos drivers da placa de rede sem fios para o mais recente possível, de modo a que as últimas normas sejam utilizadas.



Porque é que preciso do Service Pack 2 no Windows XP?

O Service Pack é um conjunto de correcções e actualizações para o sistema operativo. Nesta actualização é oferecido um mecanismo intuitivo para configurar a placa de rede sem fios, bem como, as mais recentes actualizações das normas utilizadas na ligação aos pontos de acesso eduroam, nomeadamente o 802.1x.



Porque é que só preciso inserir o meu login/password uma vez?

Após o primeiro sucesso de autenticação na rede sem fios, o sistema operativo Windows XP, armazena automaticamente as credenciais (login/password) da conta. Nos acessos seguintes o login e password não voltam a ser pedidos. Se quiser alterar o seu login/password pode consultar o link abaixo para limpar as credenciais da cache do registo do Windows XP.

Para mais informações pode consultar este link: http://support.microsoft.com/default.aspx?scid=kb;en-us;823731



VPN
O que é uma rede VPN?

Uma rede VPN é uma rede privada virtual que possibilita o acesso à rede wireless da ESAC a partir de uma ligação à Internet.

As VPN's utilizam tecnologias de segurança que garantem a autenticação, a confidencialidade e a integridade nas comunicações.

A ESAC disponibiliza aos seus utilizadores uma solução de VPN.
